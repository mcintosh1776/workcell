# PR Review Evidence Note

This documentation maintenance note defines the evidence reviewers should inspect for a Workcell lifecycle smoke PR.

Required PR evidence:

- the PR body names the Workcell repo target and branch target
- the changed-file list matches the documented task scope
- validation output is attached outside this repository document
- QA review checks the PR artifact and validation artifact before security review
- Sentinel review starts only after QA returns ready for handoff

This note is about repository review workflow evidence only. It does not assert Workcell runtime product behavior.
