# Iris Handoff: Build And CLI Usability Review

## Role

Review-only.

## Assignment

Review whether a new contributor can understand and reproduce the Workcell lab
setup without ARX context.

## Required Reading

- `README.md`
- `BUILD.md`
- `docs/lab-host-bootstrap.md`
- `docs/lab-host-lifecycle.md`
- `docs/deployment-targets.md`
- `docs/cli-ux.md`

## Review Focus

- first-run clarity
- whether Hetzner is documented as an example rather than a hard dependency
- whether "do not install this on your workstation" is clear without sounding
  hostile
- whether the CLI examples are memorable
- whether `fake`, `podman-smoke`, and `incus-smoke` profile names make sense
- whether contributors know what to do after `workcell_lab_preflight=ok`

## Output Format

- must-fix docs before wider sharing
- wording improvements
- missing examples
- later docs polish
